## Marcs-Bank

Descrição: Banco sem interface gráfica em que você pode:
1 - Criar uma conta;
2 - Efetuar um saque;
3 - Efetuar um depósito;
4 - Efetuar uma transferência de uma conta para outra;
5 - Listar todas as contas cadastradas.
